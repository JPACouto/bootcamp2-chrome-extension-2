⏱️ Pomodoro Timer

Extensão para Chrome que implementa a técnica Pomodoro (25 min foco, 5 min pausa).

Como instalar

  1. Baixe o projeto (Release no GitHub).
  2. Acesse chrome://extensions.
  3. Ative o modo desenvolvedor.
  4. Clique em Carregar sem compactação e selecione a pasta da extensão.

Permissões usadas

  • storage
  • alarms
  • notifications
